# TypingGame : タイピングゲーム

## 説明

画面にアルファベットが出てきて,そのアルファベットが書いてあるキーを押していくゲーム


## 内部

5回正しいキーを打ったら終了
正しいキーを打つたび,score++

スタート画面は無し


## GUI

背景白

### 表示させるもの

- ランダムに表示されるアルファベット
- 経過時間
- スコア(正しく打ち込んだ回数)

### クリア画面

- "Game Clear"
- time
- score


## design

- 画面中央にランダムなアルファベット
- ランダムなアルファベットの下に経過時間
- 経過時間の下にスコア



## how to play

1. dowonload TypingGame.class
1. run
```
java TypingGame
```
